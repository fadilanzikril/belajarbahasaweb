Pengenalan dan pelatihan dasar HTML5

Pengenalan tentang website dan browser, Dasar-Dasar HTML termasuk struktur HTML5, Paragraf pada HTML dan Teks serta Font, Pengaturan CSS untuk Teks dan Paragraf, Penggunaan Heading dan Emphasis,Memanfaatkan List,Hyperlink pada HTML, Pembuatan Tabel HTML, dan Ilustrasi (Gambar, Foto, dan Desain) di dalam HTML.

Implementasi Cascading Style Sheet (CSS)

Pelatihan ini bertujuan Halaman landing page yang menarik; Template atau tema blog; Mengubah PSD (Desain web) menjadi HTML; dan sebagainya. Dengan CSS, kita dapat mengatur dan mengubah layout (tata letak), warna, font, garis, dan lain-lain. Setelah mempelajari Course ini, kalian diharapkan mampu mengimplementasikan segala cara kerja CSS yang sudah dijabarkan.

Pengenalan JavaScript

Course ini akan mengajarkan Anda kode-kode Javascript secara praktis dengan setiap bab dilengkapi screencast yang mengajarkan sintaks, fungsi, dan aturan pemrograman. Mengenal Javascript, Pemilihan Editor yang Tepat Tag, Elemen, dan Atribut,Berbagai Cara Penulisan Javascript dan HTML, Berbagai Cara Menampilkan Hasil Javascript, dan lain-lain.

Bangun Situs Web yang Responsif dengan HTML dan CSS

Mendesain situs web yang menarik profesional Cara membuat situs web berfungsi di setiap perangkat seluler yang memungkinkan (desain responsif) Cara menggunakan komponen seperti membaca dokumentasi, debugging, dan menggunakan alat profesional Cara menemukan dan menggunakan aset desain gratis seperti gambar, font, dan ikon

Belajar Web Development Menggunakan Bahasa Pemrograman PHP

Menguasai dasar-dasar pemrograman web menggunakan bahasa pemrograman PHP, Menguasai dasar-dasar menggunakan database MySQL dalam pemrograman web, Menguasai cara membuat halaman web menggunakan HTML dan CSS, Menguasai cara menggunakan framework Bootstrap dalam membuat antarmuka halaman web, Menguasai teknik CRUD dalam pemrograman PHP.
